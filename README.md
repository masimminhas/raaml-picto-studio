## Getting Started

### 1. Install via Update Site
Help → Install New Software → Add:
https://masimminhas.github.io/raaml-picto-studio/

### 2. Import Example Models
Download `raaml-picto-studio-examples.zip` from [Releases](../../releases).
File → Import → General → Existing Projects into Workspace → Select Archive File

### 3. View Picto Visualizations
- Open the Picto view: Window → Show View → Other → Epsilon → Picto
- Click any `.picto` file in the Project Explorer
- The visualization renders automatically in the Picto view
