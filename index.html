RAAML Picto Studio

Eclipse p2 Update Site for RAAML Picto Studio – visualizations for safety analysis (FTA, FMEA, HARA, STPA, GSN) using Eclipse Epsilon Picto.

Installing via Eclipse

  1. Open Eclipse and go to Help → Install New Software…
  2. Click Add… and enter the following URL:
https://masimminhas.github.io/raaml-picto-studio/
  1. Select RAAML Picto Studio Feature and click Next.
  2. Accept the license and complete the installation.
  3. Restart Eclipse when prompted.
⚠️ Note: The update site bundles will be published here after the Eclipse export. Check the GitHub repository for progress.

Requirements

  • Eclipse IDE 2024-03 or later (tested on 2025-06 / 4.36.0)
  • Eclipse Epsilon Picto ≥ 2.0.0
  • Java 17+
